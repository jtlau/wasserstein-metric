# Wasserstein-metric
This repository provides a Jupyter notebook guide to use the Wasserstein metric to compare different 2D flow problems. 

## Requirements:
- geomloss==0.2.5
- h5py==3.7.0
- numpy==1.23.1
- pandas==1.3.3
- scipy==1.6.0
- torch==1.13.1

Different versions may work, but the above are the versions that worked for me.
